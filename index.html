Common CSS Properties Cheat Sheet

This web-page is intended to give an overview of four common cascading-style-sheet (CSS) properties. CSS is the language used to style websites and the current production-standard is CSS 3.

Property Name Property Layout Property Purpose
Color color: desired colour The purpose of this property is to set the colour of the font relating to the element of the selector.
Font Size font-size: size; The purpose of this property is to set the font-size of the element relating to the selector.
Text Align text-align: alignment; The purpose of this property is to set the alignment of the text of the element relating to the selector.
Font-Family font-family: font name The purpose of this property is to set the font- style of the element relating to the selector. In addition to this, if the name of the font extends to two or more words then it should be encased in "insert-font-name-here"
